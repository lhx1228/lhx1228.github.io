windows 发展史

大背景

① IBM，康柏等公司 开始生产，出售PC机

② Unix 操作系统 非常昂贵 大约四万美元

③ Unix操作系统 与Intel CPU 不兼容，所以当时都使用一个叫做CP/M操作系统，售价大约三万美元。

IBM，康柏等公司生产出来PC机，没有操作系统使用或者操作系统太贵，导致价格太过昂贵，销售不出去。

这些公司也很着急呀，而就在这时，比尔盖茨从他的朋友那听说，西雅图公司中的一个程序员仿照CP/M 开发了一个操作系统，可以运行在PC机上。

DOS

于是比尔盖茨利用其母亲的权力(比尔盖茨母亲，–》 IBM董事会董事 )找到IBM公司的董事长，跟他说：我这有一个系统，可以运行在PC机上，我可以卖给你，只需要几美元。(需要注意的是，比尔盖茨此时还并没有这个操作系统)

IBM董事长看在其母亲的面子上，以及他们也确实需要这么一个操作系统，就答应了。

同时，比尔盖茨买断系统(五万美元)，交易达成，之后改名为DOS(磁盘操作系统).

之后PC机卖出了百万台，比尔盖茨也因此获得千万美元得利润

windows1.0（1985年11月20日发布）

乔布斯登场

说个题外话：

为什么苹果公司的图标为咬了一口的苹果呢？

（乔布斯非常崇拜图灵，而图灵呢 好像是个同性恋，在当时被囚禁了，然后图灵羞愤之下咬了一口图了氰化钾的苹果，死了）

所以乔布斯就将其公司图标设为了咬了一口的苹果。

1973年，施乐旗下的阿尔托研究中心研究出一个产品，名为——Alto,它设计由一个人使用，有键盘和显示器，图形用户界面，以太网等等，对，这就是世界上具有真正意义的首台PC。现在我们都已经知道了个人电脑的重要性，但那个年代，所谓的计算机都是那种IBM的大型机、小型机，施乐高管也无法想象出普通个人怎么用这么个玩意，所以这么一伟大产品就一直处于闲置状态。

然而戏剧性的节目开始上演了，1979年(不知不觉Alto诞生6年了然而也闲置了6年)，乔布斯听说阿尔托研究中心有很多炫酷的玩意，决定到此一游。(至于为何乔布斯可以去人家的研究中心游走，原因有很多这里不赘述了)，乔布斯被允许带着一位技术人员参观，觉得参观的不过瘾后来又带着技术主管，高层人员参观了一次。 显然，乔布斯被Alto身上的图形用户界面等等技术震撼住了，参观后回到苹果公司就指导技术人员对于图形用户界面的研发，并计划将它用到丽萨电脑项目上(丽萨就是乔布斯的私生女名字)，当然，研发过程中乔布斯还狂挖施乐的技术人员，虽然丽萨的项目最后失败了，但后来的麦金塔项目继承并完善了丽萨中的图形用户界面技术并加入了鼠标。

接下来比尔盖茨先生登场了，1981年，乔布斯邀请比尔盖茨参观麦金塔样机，为即将推出的麦金塔电脑研发应用软件，在看演示时，盖茨的心情跟当初乔布斯参观施乐时是一样一样的，也被炫酷的图形用户界面和灵敏的鼠标震住了，这时盖茨也打起了自己的算盘(这俩货还真挺像的):这东西一旦上市，分分钟能灭掉我的DOS啊,图形用户界面就是未来的发展方向！

不过乔布斯看出了盖茨的心思，他担心微软会把在给苹果编写软件过程中学到的东西，用到为IBM PC所开发的应用软件上(那时候苹果的主要竞争对手就是IBM)，所以乔布斯强迫盖茨签下协议，承诺微软不会把为苹果编写的软件用于任何非苹果公司制造的计算机上。可是乔布斯忽略了一个点：该协议没有禁止微软编写类似麦金塔的操作系统，与苹果直接展开竞争。盖茨了解到乔布斯的麦金塔操作系统效法施乐，于是釜底抽薪，也从施乐直接挖人展开自己图形用户界面操作系统的研发(求此时的施乐高管心理阴影面积……)，于是，微软在跟苹果开发应用软件的同时，自己也在研发属于自己的操作系统(也就是Windows)，微软把Windows的研发放在第一的位置，而把该给苹果干的活放在了最后，结果耽误了麦金塔的发布，这让乔布斯很恼火。

后来，乔布斯发现了Windows操作系统跟麦金塔很相似,立马火冒三丈(你能想象当时乔帮主的心情吗)，他吼着让下属把盖茨找来。

盖茨一进办公室，就被10来个苹果员工包围起来，乔布斯对他大吼：你偷我们的！！！。盖茨就在这种场合说了那句”只因为你在我之前破门而入施乐公司的房间，拿走了电视，并不能意味着我后来就不能进入拿走立体声音响”。盖茨之所以当时有底气，是因为他手里有苹果公司急需的热门应用软件。

乔布斯当然不会就这么善罢甘休，他让律师起诉微软。可是祸不单行，乔布斯后来与公司总裁约翰・斯库利发生内斗，是的，就是在此时乔帮主被踢出苹果公司的。

也就在乔布斯被踢出后(85年5月)，比尔盖茨发布了Windows1.0（1985年11月20日）

微软Windows系统的第一个版本最重要的成绩就是它将图形用户界面和多任务技术引入了桌面计算领域。它用窗口替换了命令提示符，整个操作系统变得更有组织性，屏幕变成了虚拟桌面，一切都非常直观。

Windows 1.0 自带了一些简单的应用程序，包括日历、记事本、计算器等等。

其实两个人都抄袭了施乐公司。

但是，Windows系统 有了图形操作系统后，经常蓝屏，那么为什么还有那么多人买呢？

因为苹果的 电脑和系统是一起买的，不能单独买，而比尔盖茨只是买系统，所以…又赚了个盆满钵满。

Windows2.0（1987年12月9日发布）

1987年微软发行Windows 2.0，比起上一版本较受欢迎。主要原因是微软发行“运行时期版本”的Excel 和 Word,用户们还可以将应用程序的快捷方式放在桌面上，同时还引进了全新的键盘快捷键功能。

Windows3.X(1990年5月22日发布)

微软在1990年发行Windows 3.0非常成功。微软发布的第三个版本的Windows展现了一个全新的外观和三维触摸感。在这个版本中，著名的纸牌游戏Solitaire第一次登场了。微软改变了软件的架构，释放了更多的内存，并让它可以在不同类型的处理器上运行。

Windows 3.1发行于1992年3月18日，添加了对声音输入输出的基本多媒体的支持和一个CD音频播放器，以及对桌面出版很有用的TrueType字体，并开始有了虚拟内存和屏幕保护程序，微软还提高了软件的运行速度和稳定性。

Windows NT（1993年7月27日发布）

Windows NT的创新意义体现在它是第一款32位的Windows系统。这款操作系统可以在不同类型的英特尔处理器上运行，可以同时运行多个应用 程序，最多为每个应用程序分配2GB的虚拟内存。微软承诺以后会提高该系统的稳定性，即便某个软件在运行时崩溃，也不会影响到整个系统。
　　

Windows 95 ：1995年8月24日发布

Windows 95 应该可以算是空前绝后成功的作业系统，五周就卖出了700 万套，包括微软沿用至今的Internet Explorer、资源回收桶还有开始功能表都是从Windows 95 开始的。微软甚至在Windows 95 开卖时，找来了当时的一线乐团滚石乐队演唱宣传歌曲Start me up，吸引了许多根本没用过电脑的顾客也去买了一套，从此奠定微软帝国的地位。

Windows 95问世后不久，IE首次登上历史的舞台。当时它是作为一款付费产品放在Internet Jumpstart Kit中出售的。

Windows 98:1998年6月25日发布

Windows 98附带了整合式IE浏览器，标志着操作系统开始支持互联网时代的到来。微软还重新设计了其文件管理器，增加了快速关机选项，另外还努 力简化了驱动程序升级和下载系统补丁的工作。微软甚至还增加了对TV调频器的支持，让用户们可以在电脑上看电视，那主要是为了支持微软的WebTV服务。

Windows 98 和Windows 95 在介面上没什么太大的差异，因此有人批评Windows 98 没有足够的创新，不过这还是改变不了它是个成功作业系统的事实。顺道一提，Windows 98 是最后一个以MS‑DOS 为基础的版本。

Windows 2000：2000 年2月17日

Windows 2000包含新的NTFS文件系统、EFS文件加密、增强硬件支持等新特性

Windows ME：2000 年9月14日

它增加了家庭连网功能和用于播放数字音乐 和编辑视频的软件，强化了电脑的影音播放效果，同时也是第一款拥有系统还原功能的作业系统，不过同时它也有系统容易当机的问题出现，在当时出现不少批评的声音。

Windows XP：2001 年10月25日发布

2001 年，打遍天下无敌手的Windows XP 诞生了，同时它也是微软第一个针对消费者推出，并采用Windows NT 底层设计的作业系统，不仅稳定度提升，效能表现也十分出色，最辉煌时期全世界甚至有76% 的电脑安装Windows XP 作业系统，一直到2012 年8 月才被Windows 7 超越。

Windows Vista：2006 年

Vista换上了一个更加现代化的界面，微软称之为“Aero”；还增加了一些安全功能，改善了搜索功能。微软还调整了某些内置的生产力和娱乐软件，比如邮件、日历、DVD制作和图库等。

Windows Vista 花了微软五年的时间打造，号称几乎所有市面上的电脑都能使用Vista，不过这番言论立刻被消费者打脸，英国的泰晤士报就曾报导英国只有不到一半的电脑可以跑得动Vista 华丽的介面，有不少消费者在购买新电脑时甚至还会请店家帮忙降级回Windows XP。

Windows 7：2009 年

微软修改了Windows 7中的任务条，允许用户“钉”软件以及快速浏览公开软件的预览版本。它还增加了在开放应用程序中发布流行任务的快捷键，并且可以快速组织窗口，将它们收拢到屏幕的一角。

Windows 7还增加了一些更加高级的触摸导航功能，进一步改善了搜索、通用系统性能和内置媒体播放器软件。

虽然上一代作业系统Vista 看起来没那么成功，不过介面颇为相似的Windows 7 可就出色许多，开关机的速度较前几代来得快，系统稳定度也很不错，更是难得超过Windows XP 市占率的一款作业系统。

Windows 8：2012 年

Windows 8最大的成就是将微软领入了平板电脑时代，它的界面是专为触摸式控制而设计的。其他的变化还包括微软称之为“Metro”的界面和一系列全新的触摸应用。微软还在Windows 8系统中增加了内置商店，以便用户寻找和下载新的软件。

Windows 8还带来一个开创性的变化，Windows系统第一次开始支持ARM的芯片了。

Windows 8.1：2014 年

Windows8.1重新启用了“开始”按钮

Windows 8.1增加了SkyDrive；Windows 8.1支持3D打印，增加了一个新的Xbox Music应用，完善了对多显示屏的支持。

Windows 10：2015 年
