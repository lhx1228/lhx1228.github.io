bash快捷键

移动快捷键

  • ctrl+a：

将光标移到行首。

  • ctrl+e

将光标移到行尾

  • ctrl+b

将光标向前移一格。

  • ctrl+f

将光标向后移一格。

  • Alt+b

按单词前移

  • Alt+f

按单词后移

  • crtl+xx

在命令行和行首之间移动

删除快捷键

  • ctrl+d

删除一格字符

  • ctrl+h

删除光标前的一个字符

  • alt+d

从光标删除至行尾

  • ctrl+u

从光标处删除至行首

  • ctrl+k

从光标删除至行尾

  • ctrl+w

删除一个单词

执行命令

  • ctrl+r

快捷搜索

  • ctrl+p

历史中的上一条命令

  • ctrl+n

历史中的下一条命令

其他

  • ！！

执行上一条命令

  • cd -

回到上一个工作目录。

vim打开多个文件

  • 第一种方法(在命令行)

vim file1 file2

  • 第二种方法(已打开一个文件)

:open file2

  • 多个文件之间切换

:bn 切换到下一个文件

:bp 切换到上一个文件
