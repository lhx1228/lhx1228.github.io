将命令序列的输出读入变量

 
有时我们在使用linux执行某些操作时，需要很多条命令，一条一条输入未免太过复杂。接下来介绍一下将命令组合的技巧，一个命令的输出可以作为另一个命令的输入，而这个命令的输出又会传递至另一个命令，依次类推：
 
1. 先从组合两个命令开始:

ls | cat -n > out.txt 
ls 的输出（当前目录内容的列表）被传给 cat -n ，后者将通过 stdin 所接收到输入内容加上行号，然后将输出重定向到文件 out.txt 。

 
2.如果要输出这些组合命令的结果，我们使用以下方法：

cmd=$(ls | cat -n)
echo $cmd
或者
cmd=`ls | cat -n`
echo $cmd

 
3.使用()操作符来定义一个子shell

我们可以使用以下格式来自由组合命令，例如：
（cd /sbin;ls)
(cd ofenuse;ls;vi test.sh)

read的一些实用操作

 
很多时候，我们在输入数据时，以回车来结束输入是不符合实际的，或者是我们不希望我们的输出结果回显等等，接下来就来介绍一些简单技巧：
 

  • 读取n个字符之后停止继续读入

    read -n number_of_chars variable_name
例如：
read -n 2 var
echo $var

     

  • 无回显

    read -s var

     

  • 显示提示信息

    read -p "Enter input:" var

     

  • 在一定的时间内读取输入

    read -t time cvar
例如：
read -t 2 car
#在2秒内将键入的字符串读入变量var

  • 用特定的定界符作为输入的结束

    read -d delim_char var
例如：
read -d ":" var
$ hello:

比较与测试

1. [ condition ] && action; # 如果 condition 为真，则执行 action ；

2.[ condition ] || action; # 如果 condition 为假，则执行 action 。
 

  • 算术比较

条件通常被放置在封闭的中括号内。一定要注意在 [ 或 ] 与操作数之间有一个空格。如果
忘记了这个空格，脚本就会报错。

对变量或值进行算术条件判断：

[ $var -eq 0 ] #当 $var 等于 0 时，返回真

[ $var -ne 0 ] #当 $var 为非 0 时，返回真

其他重要的操作符如下：

 -gt ：大于。
 -lt ：小于。
 -ge ：大于或等于。
 -le ：小于或等于。
